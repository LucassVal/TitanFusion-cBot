10. Troubleshooting

Dashboard shows "--" for all values

Wait for the indicator to calculate. The first few bars may show placeholder values until enough data is available.

Cloud appears even when disabled

Make sure to set the "Show X Cloud" parameter to false in the indicator settings. The indicator now properly respects all cloud visibility settings.

Signals not appearing

Check that:

  • MTF alignment is sufficient (at least 3/4)
  • You're not in dead trading hours
  • Volatility is within normal range

Build errors

Ensure you're using the latest version of cTrader (4.0+) and that all files are properly copied to the indicators folder.
